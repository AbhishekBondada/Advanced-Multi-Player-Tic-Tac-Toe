# Advanced Multi-Player Tic-Tac-Toe

An advanced Tic-Tac-Toe web application built using HTML, CSS, and JavaScript. This project supports multiple board sizes, score tracking, game history, winner highlighting, and responsive design.

---

## Features

* 3 × 3 Standard Tic-Tac-Toe Mode
* 4 × 4 Advanced Tic-Tac-Toe Mode
* Real-Time Turn Indicator
* Scoreboard Tracking
* Winner Detection
* Draw Detection
* Winning Cell Highlighting
* Match Reset Functionality
* Next Round Functionality
* Game History Tracking
* Responsive User Interface
* Modern Dark Theme Design

## Game Rules

### 3 × 3 Mode

* Two players take turns placing X and O.
* The first player to complete a row, column, or diagonal wins.

### 4 × 4 Mode

* Players must complete an entire row, column, or diagonal of four identical symbols.
* The first player to do so wins the round.

---

## Application Workflow

1. Select the board size.
2. Player X starts the game.
3. Players alternate turns.
4. The system checks for a winner after every move.
5. If a winner is found:

   * Winning cells are highlighted.
   * Scoreboard is updated.
   * History is recorded.
6. If all cells are filled:

   * Match is declared a draw.
7. Players can:

   * Start the next round.
   * Reset the entire match.
